## ✨ 核心功能

### 1️⃣ 快速场景按钮（一键运行）

点击任意按钮即可运行预设场景：

| 按钮 | 功能 | 耗时 |
|-----|------|------|
| 🎬 **快速测试** | 30次运行，5%丢包 | 30秒 |
| 📊 **基线对比** | 对比所有4种防御 | 2分钟 |
| 📉 **丢包测试** | 10%丢包率测试 | 2分钟 |
| 🔀 **乱序测试** | 30%乱序率测试 | 2分钟 |
| ⚡ **恶劣网络** | 15%丢包+30%乱序 | 2分钟 |
| 🎯 **选择性攻击** | 针对UNLOCK命令 | 2分钟 |

**使用方法：** 直接点击按钮，等待结果！

---

### 2️⃣ 自定义实验（滑动条调整）

**防御机制选择：**
- 单选按钮，点击即可选择
- 5种选项：全部对比、无防御、滚动计数器、滑动窗口、挑战响应

**运行次数：**
- 滑动条调整（10-200次）
- 实时显示当前值
- 推荐：50-100次

**丢包率：**
- 滑动条调整（0.0-0.5）
- 0.0 = 无丢包
- 0.1 = 10%丢包
- 0.2 = 20%丢包

**乱序率：**
- 滑动条调整（0.0-0.5）
- 0.0 = 无乱序
- 0.3 = 30%乱序

**运行按钮：**
- 点击"▶️ 运行自定义实验"
- 自动开始运行
- 实时显示进度

---

### 3️⃣ 其他功能

**📊 生成图表：**
- 一键生成所有5张图表
- 保存到 `figures/` 目录

**📋 导出表格：**
- 导出实验数据为表格
- 保存到 `docs/metrics_tables.md`

**🗑️ 清空输出：**
- 清除右侧输出窗口
- 方便开始新实验

## 📊 界面元素说明

### 左侧控制面板

**快速场景区：**
- 6个彩色按钮
- 每个按钮对应一个预设场景
- 点击即可运行

**自定义实验区：**
- 防御机制：5个单选按钮
- 运行次数：滑动条（10-200）
- 丢包率：滑动条（0.0-0.5）
- 乱序率：滑动条（0.0-0.5）
- 运行按钮：绿色大按钮

**其他功能区：**
- 生成图表：蓝色按钮
- 导出表格：紫色按钮
- 清空输出：灰色按钮

### 右侧输出面板

**实时输出窗口：**
- 黑色背景，白色文字
- 显示所有运行输出
- 自动滚动到最新内容
- 可以用鼠标滚轮查看历史

**状态栏：**
- 显示当前状态
- "就绪" = 可以运行新实验
- "运行中" = 实验正在进行

## 🔧 高级功能

### 多实验管理

**注意：** 
- 一次只能运行一个实验
- 如果点击第二个实验，会提示"已有实验正在运行"
- 等待当前实验完成后再运行下一个

### 输出管理

- 所有输出都会累积显示
- 点击"清空输出"可以清除历史
- 输出窗口支持复制粘贴
